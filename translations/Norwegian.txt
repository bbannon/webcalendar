# Formatet består av en valgfri overskrift:
#
# Oversatt av: navnet ditt og e-postadressen din
# Siste oppdatering: datoen for oversettelsen
# (Inkludert de ledende "#" og eventuelle kommentarer fra tidligere oversettere.
# Den FØRSTE tomme linjen avslutter disse overskriftskommentarene.)
#
# Etterfulgt av så mange som nødvendig:
#
# Engelsk setning: oversettelse
# (En til en linje. UTEN den ledende "#".
# Den "engelske setningen:", inkludert kolon ":", er levert av
# WebCalendar-utviklere. Din del er "oversettelsen".)
#
# Verktøykatalogen inneholder to perl-skript som hjelper deg:
# - check_translation.pl
# indikerer om all tekst er oversatt.
# - update_translation.pl
# vil analysere programfilene for å finne alle translate ('frase') og
# verktøytips ('frase'). Det vil ordne setningene i henhold til siden
# (fil) som inneholder den første forekomsten. Eventuelt kan den merke
# setninger som ennå ikke er oversatt ved å sette
#  << MISSING >>
# på linjen over uttrykket. Og hvis uttrykket er en forkortelse,
# den inkluderer den fulle engelske teksten på linjen under uttrykket.
#
# Merk: Jeg prøver å kutte ned på den totale filstørrelsen. Hvis "oversettelsen"
# word / phrase er identisk på ditt språk og engelsk, bruk bare et "=" tegn.
#
# Merk: Du vil legge merke til oppføringen for "May_" under. Det er en grunn til dette.
#  Oversette "May_" til hele månedsnavnet og "May" til,
# vanligvis tre bokstaver, måned forkortelse (som "oktober" for "oktober").
