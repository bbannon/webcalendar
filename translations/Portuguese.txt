#
# Traduzido por: seu nome e e-mail
# Última atualização: a data da tradução
# (Incluindo o "#" inicial e quaisquer comentários feitos por tradutores anteriores.
# A PRIMEIRA linha em branco termina esses comentários de cabeçalho.)
#
# Seguido por quantos forem necessários:
#
# Frase em inglês: tradução
# (Um para uma linha. SEM o "#" inicial.
# A "frase em inglês:", incluindo os dois pontos ":", é fornecida pelo
# Desenvolvedores WebCalendar. Sua parte é a "tradução".)
#
# O diretório de ferramentas contém dois scripts perl para ajudá-lo:
# - check_translation.pl
# indica se todo o texto foi traduzido com sucesso.
# - update_translation.pl
# irá analisar os arquivos do programa para encontrar todas as traduções ('frase') e
# dica de ferramenta ('frase'). Irá organizar as frases de acordo com a página
# (arquivo) que contém a primeira ocorrência. Opcionalmente, ele pode marcar o
# frases que ainda não foram traduzidas colocando
#  << MISSING >>
# na linha acima da frase. E, se a frase for uma abreviatura,
# incluirá o texto completo em inglês na linha abaixo da frase.
#
# Nota: Estou tentando reduzir o tamanho geral do arquivo. Se a "tradução"
# palavra / frase é idêntica em seu idioma e em inglês, basta usar um sinal "=".

#  Traduzir "May_" para o nome completo do mês e "May" ao,
