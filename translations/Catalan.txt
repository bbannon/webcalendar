# El format consisteix en una capçalera opcional:
#
# Traduït per: el vostre nom i correu electrònic
# Última actualització: data de la traducció
# (Incloent el "#" principal i qualsevol comentari fet per traductors anteriors.
# La PRIMERA línia en blanc finalitza aquests comentaris de capçalera.)
#
# Seguit de tants com calgui:
#
# Frase en anglès: traducció
# (Un a una línia. SENSE el "#" principal.
# La "frase en anglès:", inclosos els dos punts ":", la proporciona el
# Desenvolupadors de WebCalendar. La vostra part és la "traducció".)
#
# El directori d'eines conté dos scripts perl per ajudar-vos:
# - check_translation.pl
# indica si tot el text s'ha traduït correctament.
# - update_translation.pl
# analitzarà els fitxers del programa per trobar tots els traductors ('frase') i
# descripció emergent ("frase"). Organitzarà les frases segons la pàgina
# (fitxer) que conté la primera ocurrència. Opcionalment, pot etiquetar el fitxer
# frases que encara no es tradueixen posant
#  << MISSING >>
# a la línia que hi ha a sobre de la frase. I, si la frase és una abreviatura,
# inclourà el text complet en anglès a la línia de sota de la frase.
#
# Nota: Intento reduir la mida global del fitxer. Si la "traducció"
# paraula / frase és idèntica en el vostre idioma i en anglès, només cal que utilitzeu un signe "=".
#
# Nota: notareu l'entrada següent per a "Maig_". Hi ha una raó per això.
# Tradueix "May_" al nom del mes complet i "May" al,
# normalment, abreviatura de tres lletres i mesos (com ara "Oct" per a "Octubre").
#
# He utilitzat Google Translate per a això.
# Ha de ser suficient fins que un parlant nadiu ho pugui verificar.
# Bruce Bannon <wcDev@bbannon.net>
#
