#
# Översatt av: ditt namn och e-post
# Senaste uppdatering: datum för översättning
# (Inklusive det ledande "#" och eventuella kommentarer från tidigare översättare.
# Den FÖRSTA tomma raden avslutar dessa rubrikkommentarer.)
#
# Följt av så många som behövs:
#
# Engelska frasen: översättning
# (En till en rad. UTAN det ledande "#".
# Den "engelska frasen:", inklusive kolon ":", tillhandahålls av
# WebCalendar-utvecklare. Din del är "översättningen".)

# Verktygskatalogen innehåller två perl-skript som hjälper dig:
# - check_translation.pl
# indikerar om all text har översatts.
# - update_translation.pl
# analyserar programfilerna för att hitta alla översättningar ('fraser') och
# tooltip ('fras'). Det ordnar fraserna enligt sidan
# (fil) som innehåller den första förekomsten. Alternativt kan den märka
# fraser som ännu inte översatts genom att sätta
#  << MISSING >>
# på raden ovanför frasen. Och om frasen är en förkortning,
# den kommer att innehålla den fullständiga engelska texten på raden under frasen.
#
