# Oversat af: dit navn og din e-mail
# Sidste opdatering: datoen for oversættelsen
# (Inklusive de førende "#" og eventuelle kommentarer fra tidligere oversættere.
# Den FØRSTE tomme linje afslutter disse overskriftskommentarer.)
#
# Efterfulgt af så mange som nødvendigt:
#
# Engelsk sætning: oversættelse
# (En til en linje. UDEN det førende "#".
# Den "engelske sætning:", inklusive kolon ":", leveres af
# WebCalendar-udviklere. Din del er "oversættelsen".)
#
# Værktøjskataloget indeholder to perl-scripts, der kan hjælpe dig:
# - check_translation.pl
# angiver, om al tekst er oversat.
# - update_translation.pl
# analyserer programfilerne for at finde alle oversætte ('sætning') og
# værktøjstip ('sætning'). Det arrangerer sætningerne i henhold til siden
# (fil), der indeholder den første forekomst. Eventuelt kan den tagge
# sætninger, der endnu ikke er oversat ved at sætte
#  << MISSING >>
# på linjen over sætningen. Og hvis sætningen er en forkortelse,
# det inkluderer den fulde engelske tekst på linjen under sætningen.
#
# Bemærk: Jeg forsøger at skære ned på den samlede filstørrelse. Hvis "oversættelsen"
# ord / sætning er identisk på dit sprog og engelsk, brug bare et "=" tegn.
#
# Bemærk: Du vil bemærke posten til "May_" nedenfor. Der er en grund til dette.
# Oversæt "May_" til hele månedsnavnet og "May" til,
