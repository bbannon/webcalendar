# Dutch nl
#
# Het formaat bestaat uit een optionele header:
#
# Vertaald door: uw naam en e-mailadres
# Laatste update: de datum van vertaling
# (Inclusief de leidende "#" en eventuele opmerkingen van eerdere vertalers.
# De EERSTE lege regel beëindigt deze koptekstopmerkingen.)
#
# Gevolgd door zoveel als nodig:
#
# Engelse zin: vertaling
# (Een op een regel. ZONDER de eerste "#".
# De "Engelse zin:", inclusief de dubbele punt ":", wordt geleverd door de
# WebCalendar-ontwikkelaars. Uw deel is de "vertaling".)
#
# De directory tools bevat twee perl-scripts om u te helpen:
# - check_translation.pl
# geeft aan of alle tekst succesvol is vertaald.
# - update_translation.pl
# zal de programmabestanden ontleden om alle vertaalde ('zin') en
# tooltip ('zin'). Het rangschikt de zinnen volgens de pagina
# (bestand) dat het eerste exemplaar bevat. Optioneel kan het de
# zinnen die nog niet zijn vertaald door
#  << MISSING >>
# op de regel boven de zin. En als de zin een afkorting is,
# het bevat de volledige Engelse tekst op de regel onder de zin.
#
# Opmerking: ik probeer de totale bestandsgrootte te verkleinen. Als de "vertaling"
# woord / zin is identiek in uw taal en Engels, gebruik gewoon een "=" teken.
#
# Opmerking: u zult de vermelding voor "May_" hieronder opmerken. Daar is een reden voor.
# Vertaal "May_" naar de volledige naam van de maand en "May" naar de,
# gewoonlijk drieletterige afkorting van de maand (zoals "okt" voor "oktober").
