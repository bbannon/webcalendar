# Englanninkielinen lause: käännös
# (Yksi riville. ILMAN etureunaa "#".
# "Englanninkielinen lause:", mukaan lukien kaksoispiste ":", tarjoaa
# WebCalendar-kehittäjää. Sinun osasi on "käännös".)
#
# Työkaluhakemisto sisältää kaksi perl-komentosarjaa, jotka auttavat sinua:
# - check_translation.pl
# osoittaa, onko kaikki teksti käännetty onnistuneesti.
# - update_translation.pl
# jäsentää ohjelmatiedostot löytääksesi kaikki käännökset ('lause') ja
# työkaluvinkki ('lause'). Se järjestää lauseet sivun mukaan
# (tiedosto), joka sisältää ensimmäisen esiintymän. Vaihtoehtoisesti se voi tunnistaa
# lausetta, joita ei ole vielä käännetty laittamalla
#  << MISSING >>
# rivillä lauseen yläpuolella. Ja jos lause on lyhenne,
# se sisältää kokonaisen englanninkielisen tekstin lauseen alla olevalla rivillä.
#
# Huomaa: Yritän pienentää tiedostojen kokoa. Jos "käännös"
# sana / lause on identtinen kielelläsi ja englannissasi, käytä vain "=" -merkkiä.
#
# Huomaa: Huomaat merkinnän "May_" alla. Tähän on syy.
# Käännä "May_" koko kuukauden nimeksi ja "May"
# yleensä kolmen kirjaimen, kuukauden lyhenne (kuten "lokakuu" sanoille "lokakuu").
